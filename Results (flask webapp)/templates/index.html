Project Overview

This project develops a system that predicts the safest routes on maps by analyzing criminal activities along different paths. The system is particularly valuable for:

  • Night-time travel safety
  • Secure courier deliveries
  • Tourist navigation in unfamiliar areas
  • Personal safety applications

We implemented four distinct machine learning approaches to solve this problem, each with different strengths and characteristics.
